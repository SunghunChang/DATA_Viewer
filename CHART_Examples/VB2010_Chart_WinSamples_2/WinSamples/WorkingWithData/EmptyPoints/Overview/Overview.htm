Empty points are data points that do not have Y values or have been marked as empty. They are visually displayed in the chart area. Empty points are not the same as missing data points, which  refer to data points that are missing in a data set.

Empty Points Usage

Use empty points to:

  • Represent points with NULL values.
  • Change the appearance of missing data points.
  • Align two or more series.

Empty data points can be added to a series using:

  • The Empty property of each data point in a series. If Empty is set to True, a point is marked as empty.
  • Data binding to DBNull values. If a point is data bound to a DBNull value it will be marked as empty.
  • The InsertEmptyPoints method of the DataManipulator class, which checks for missing data and inserts empty points into a series using intervals.
